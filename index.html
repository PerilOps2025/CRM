Google Sheets Setup Instructions

×
Step 1: Create a new Google Sheet with the following columns in row 1:

A: Name | B: Email | C: Phone | D: Company | E: Status | F: Source | G: Notes | H: Date Added
Step 2: Make your Google Sheet public:
• Click "Share" in the top right
• Click "Change to anyone with the link"
• Set to "Viewer" permissions
• Copy the sharing link
Step 3: Get a Google Sheets API Key:
• Go to Google Cloud Console
• Enable Google Sheets API
• Create credentials (API Key)
• Copy the API key
Step 4: Enter your Sheet URL and API Key above and click "Connect to Google Sheets"
